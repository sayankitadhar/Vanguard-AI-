# Vanguard AI - Anti-Gravity UI

This project implements the "Anti-Gravity" UI layout for the Vanguard AI home page using React, Framer Motion, and Vanilla CSS.

## Features

- **Anti-Gravity Cards**: Floating effect with staggered timing.
- **Physics-based Interaction**: Cards stop drifting and tilt in 3D toward the cursor on hover.
- **Parallax Background**: Nebula and Grid layers move at different speeds on scroll.
- **Visuals**: Glassmorphism, Neon Cyan glows, pulsing Green Heart logo.

## Setup & Run

1. `npm install`
2. `npm run dev`

## Tech Stack

- **Framework**: Vite + React + TypeScript
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Styling**: Vanilla CSS (CSS Modules/Global Styles) - No Tailwind used.
